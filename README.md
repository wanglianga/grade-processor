# 学校成绩单离线处理器

基于 Electron 的桌面应用，支持导入多个课程成绩表、学生名单和课程学分表，在本机完成成绩合并、异常检测和报表导出。

## 原始需求

> 做一个学校成绩单离线处理器，Electron 桌面窗口接收多个课程成绩表、学生名单和课程学分表，在本机做成绩合并、资格判断和表格输出。核心字段有学号、姓名、班级、课程、平时分、期末分、总评、学分、缺考、作弊、缓考和任课教师。工具要识别学号不存在、同课重复成绩、总评分计算不一致、缺考但有分数、作弊不得补考和缓考单独列出。教务老师能导出补考名单、成绩异常清单和班级汇总。

## 功能特性

- **多源导入**：支持同时导入多个课程成绩表（Excel/CSV），学生名单和课程学分表
- **字段映射**：自动识别中文表头（如"平时成绩"→"平时分"，"课程名称"→"课程"等）
- **成绩合并**：自动将成绩数据与学生名单和学分表合并补全
- **异常检测**：
  - 学号不在学生名单中
  - 同一学生同一课程存在重复成绩
  - 总评成绩与按公式计算的值不一致
  - 标记缺考但仍存在分数
  - 作弊标记（不得补考）
  - 缓考单独列出
- **报表导出**：
  - 补考名单（不及格且非作弊、非缓考）
  - 成绩异常清单
  - 缓考名单
  - 班级汇总（含平均分、最高分、最低分、及格率）
- **权重配置**：可自定义平时分与期末分权重（默认 3:7）

## 启动方式

### 前置要求

- Node.js 18+
- npm

### Docker 一键启动（推荐）

#### 前置要求

- Docker
- Docker Compose

#### 启动步骤

```bash
docker compose up --build
```

后台运行：

```bash
docker compose up --build -d
```

停止服务：

```bash
docker compose down
```

> 注意：Docker 模式下使用 Xvfb 虚拟显示运行 Electron，适用于无显示器的服务器环境。桌面使用建议采用本地启动方式。

### 本地启动

#### 1. 安装依赖

```bash
npm install
```

#### 2. 启动应用

```bash
npm start
```

应用启动后自动弹出桌面窗口。

## 使用说明

1. **导入数据**：在左侧面板分别导入课程成绩表、学生名单、课程学分表（支持 .xlsx/.xls/.csv）
2. **配置权重**：调整平时分与期末分权重（默认 0.3 和 0.7）
3. **点击处理**：点击"开始处理"按钮执行合并和异常检测
4. **查看结果**：在右侧标签页切换查看合并成绩、异常检测、补考名单、缓考名单、班级汇总
5. **导出报表**：点击各标签页的"导出"按钮保存为 Excel 文件

|------|------|------|------|--------|--------|------|------|------|------|----------|

### 学生名单

| 学号 | 姓名 | 班级 |
|------|------|------|

### 课程学分表

| 课程 | 学分 | 任课教师 |
|------|------|----------|

> 表头支持多种中文写法自动映射，如"平时成绩"→"平时分"，"课程名称"→"课程"等。
